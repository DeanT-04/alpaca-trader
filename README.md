# Alpaca Trader

A high-performance algorithmic trading system for Alpaca Markets, built with modern Python practices.

## Directory Structure

- `src/`: Source code
- `tests/`: Test suite
- `docs/`: Documentation
- `scripts/`: Devops and utility scripts

## Quick Start

1. **Install Dependencies**:
   ```bash
   make setup
   ```

2. **Run Tests**:
   ```bash
   make test
   ```

3. **Run Application**:
   ```bash
   python -m alpaca_trader
   ```
